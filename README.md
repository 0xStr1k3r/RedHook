# RedHook 🪝

Phishing simulation platform for security awareness testing.

## Quick Start

```bash
git clone https://github.com/your-repo/redhook.git
cd redhook
./setup.sh
./start.sh
```

## Requirements

- Go 1.26+
- PostgreSQL 15+
- ngrok (for public URLs)

## Full Documentation

👉 **https://redhook.0xchiru.dev**

## Setup Steps

1. Clone & setup
2. Configure PostgreSQL
3. Configure AWS SES
4. Setup ngrok
5. Access dashboard at http://localhost:8080

Default login: admin@redhook.local / changeme123
